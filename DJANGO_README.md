# АвтоДеталь — Django-проект

Интернет-магазин автозапчастей на Django по макету и требованиям lab 6 / django task 5.

## Структура

- **store/** — настройки проекта (settings, urls, wsgi)
- **main/** — главная страница, вход, выход, регистрация
- **products/** — каталог товаров, категории, карточка товара, админка
- **cart/** — корзина (сессия или пользователь), добавление/удаление
- **templates/** — базовый шаблон и шаблоны приложений
- **static/** — CSS и JS из макета
- **media/** — загружаемые изображения товаров

## Установка и запуск

```bash
# Виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Linux/macOS

# Зависимости
pip install -r requirements.txt

# Миграции (создать и применить)
python manage.py makemigrations main products cart
python manage.py migrate

# Тестовые данные: категории, товары и демо-пользователь для входа
python manage.py seed

# Запуск
python manage.py runserver
```

Откройте: http://127.0.0.1:8000/

**Вход по демо-аккаунту** (после `python manage.py seed`):
- **Логин (email):** admin@autodetail.ru  
- **Пароль:** admin  

В поле «Email или телефон» на странице входа вводите именно **email** (admin@autodetail.ru).

- Главная: /
- Каталог: /catalog/
- Корзина: /cart/
- Вход: /login/
- Регистрация: /register/
- Админка: /admin/

## Функциональность

- Главная страница с текстом о компании (макет АвтоДеталь).
- Каталог: поиск по названию/описанию/артикулу, фильтр по категории.
- Карточка товара и страница категории.
- Корзина: для гостей — по сессии, для авторизованных — по пользователю; добавление в корзину (форма или AJAX), удаление позиции.
- Вход / выход / регистрация (Django auth).
- Админ-панель: категории и товары (в т.ч. артикул, цена, наличие, изображение).

## Зависимости

- Django >= 4.2
- Pillow (для ImageField товаров)
