# Документация интернет-магазина автозапчастей

## Общее описание структуры сайта

Сайт состоит из единого шаблона с фиксированным хэдером в верхней части экрана и футером в нижней части. Хэдер содержит навигационные элементы и контактную информацию, футер содержит информацию о копирайте. Основная область между хэдером и футером отображает контент текущей страницы.

## Структура хэдера (навигационной панели)

Хэдер фиксирован в верхней части экрана и разделен на три горизонтальные зоны:

1. **Левая зона** - содержит логотип магазина автозапчастей. Логотип является кликабельной ссылкой, ведущей на главную страницу.

2. **Центральная зона** - содержит текстовые ссылки на основные страницы сайта: Главная, Каталог, Корзина, Вход (для неавторизованных пользователей) или Профиль (для авторизованных пользователей).

3. **Правая зона** - содержит контактную информацию компании: физический адрес магазина, номер телефона для связи, время работы магазина.

## Структура футера (подвала сайта)

Футер расположен в нижней части каждой страницы и содержит строку с информацией об авторских правах, включающую название магазина и текущий год.

## Описание страниц

### Главная страница

Главная страница содержит текстовую информацию о компании. Текст структурирован по разделам: приветствие и краткое описание магазина, история компании и срок работы на рынке, описание предлагаемых товаров и услуг, преимущества сотрудничества, контактная информация и призыв к действию:

### Страница каталога товаров

Страница каталога содержит табличный список товаров. Каждая строка таблицы представляет один товар и содержит следующие колонки: идентификатор детали (артикул), изображение товара, наименование детали, текстовое описание, информация о наличии на складе, срок доставки при отсутствии на складе, цена товара.

Под описанием каждого товара находятся элементы управления: поле для ввода количества (со стрелками для увеличения/уменьшения) и кнопка "Добавить в корзину".

В верхней части страницы расположена поисковая строка с полем ввода и кнопкой "Найти". Поиск осуществляется по трем параметрам: наименованию товара (частичное совпадение), идентификатору товара (точное совпадение), описанию товара (частичное совпадение).

### Страница корзины

Страница корзины отображает список товаров, добавленных пользователем. Для каждого товара показывается наименование, количество, цена за единицу и общая стоимость позиции. Рядом с каждым товаром находятся кнопки для удаления и изменения количества.

Под списком товаров отображается расчет стоимости: сумма за все товары, стоимость доставки и итоговая сумма к оплате.

В нижней части страницы находится форма оформления заказа. Форма содержит поля для указания способа оплаты, адреса доставки и персональных данных пользователя. Для авторизованных пользователей часть данных подставляется автоматически из профиля.

### Страница входа в систему

Страница входа содержит форму с двумя полями ввода: для логина (email или телефон) и для пароля. Под полем пароля находится иконка для отображения/скрытия вводимых символов. Под полями размещен чекбокс "Запомнить меня".

Под формой находятся три кнопки/ссылки: основная кнопка "Войти" для отправки формы, ссылка "Зарегистрироваться" для перехода на страницу регистрации, ссылка "Забыли пароль" для восстановления доступа.

### Страница регистрации

Страница регистрации содержит форму с полями для ввода: ФИО, email, телефон, пароль, подтверждение пароля. Под каждым полем пароля находится иконка для отображения/скрытия вводимых символов.

Внизу формы расположен чекбокс для согласия с пользовательским соглашением и политикой конфиденциальности, а также кнопка "Зарегистрироваться". Под формой находится ссылка "Уже есть аккаунт?" для перехода на страницу входа.

### Страница истории заказов

Страница истории заказов доступна только авторизованным пользователям. Страница разделена на два основных раздела: "Текущие заказы" и "Завершенные заказы".

В каждом разделе заказы представлены в виде списка. Для каждого заказа отображается номер, дата оформления, текущий статус, состав заказа (список товаров с количеством и ценой) и общая сумма.

Статусы заказов имеют следующие значения: "В обработке", "Принято поставщиком", "Доставлен в филиал", "Выдан", "Отменен".

Для каждого заказа доступны кнопки дополнительных действий: "Подробнее" для просмотра деталей, "Отменить заказ" для текущих заказов, "Повторить заказ" для завершенных.

## Функциональные особенности

### Система аутентификации
Пользователи регистрируются с указанием email и телефона. Пароли хранятся в зашифрованном виде. После регистрации или входа пользователь получает доступ к персональной корзине, которая сохраняется между сессиями.

### Корзина покупок
Корзина привязана к учетной записи пользователя. Товары остаются в корзине после выхода из системы и доступны при последующих входах. В корзине пользователь может изменять количество товаров, удалять позиции и оформлять заказ.

### Поисковая система
Поиск осуществляется по трем критериям одновременно. При вводе текста в поисковую строку система ищет совпадения в названиях товаров (частичное совпадение), идентификаторах (точное совпадение) и описаниях товаров (частичное совпадение). Результаты отображаются в той же табличной форме, что и основной каталог.

### Административная панель
Существует предварительно созданный административный аккаунт с расширенными правами. Администратор имеет доступ к управлению всем контентом сайта: добавление, редактирование и удаление товаров, управление заказами, просмотр и управление пользователями, настройка системы.

### Оформление заказа
При оформлении заказа пользователь указывает способ оплаты, адрес доставки и подтверждает свои контактные данные. Для авторизованных пользователей часть данных подставляется автоматически. После подтверждения заказа корзина очищается, заказ появляется в истории заказов с статусом "В обработке".

### История заказов
Каждый пользователь видит только свои заказы. Заказы сортируются по дате (сначала новые). Статусы заказов обновляются администратором и отражают текущее состояние выполнения заказа.
